# ai-terminal-real-scenario

## Purpose

Regression plan for a real AI terminal plan-creation scenario.

## Flow

1. Open the local ops dashboard fixture from `resources/ops-dashboard.html`.
2. Save a screenshot as `output/screenshots/ops-dashboard.png`.
3. Extract the page title into variable `title_text`.
4. Write `title_text` to `output/text/title.txt`.

The target fixture was inspected before writing the plan. Evidence: page title `Ops Dashboard`, body text `Export risk report Risk report pending`, no login or challenge detected.
