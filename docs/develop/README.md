# 开发文档

## 按任务选择

- 初始化环境：读 [安装指南](./安装指南.md)。
- 本地启动、CLI 和常用命令：读 [运行说明](./运行说明.md)。
- AI 协作、`AGENTS.md`、资源路径和验证矩阵：读 [AI 协作开发规范](./AI协作开发规范.md)。
- 选择自检或回归命令：读 [测试与验证说明](./测试与验证说明.md)。
- 浏览器组件改动：读 [浏览器组件回归说明](./浏览器组件回归说明.md)。
- AI 终端工具改动：读 [AI 终端工具开发检查清单](./AI终端工具开发检查清单.md)。
- 需要一个完整流程样例：读 [开发流程示例](./开发流程示例.md)。
