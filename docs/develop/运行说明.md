# 运行说明

## 运行单计划

```powershell
python .\main.py --file .\examples\scenarios\basic\fill_system_account.example.json
```

## 运行顺序套件

```powershell
python .\main.py --file .\examples\suites\regression\sequential_projects.example.json
```

## 运行并发套件

```powershell
python .\main.py --file .\examples\suites\regression\parallel_projects.example.json
```

## 按标签筛选 suite

```powershell
python .\main.py --file .\examples\suites\smoke\tagged_projects.example.json --include-tags smoke,login --tag-mode all
```

## 输出目录

- 单计划：`output/plan-runs/`
- 套件：`output/suite-runs/`
