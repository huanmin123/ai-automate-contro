# 测试与验证说明

当前主要通过示例计划进行功能验证。

所有回归示例应优先使用本地 HTML 夹具，不依赖真实外部网站。

## 建议验证项

- 基础页面打开与输入
- 变量替换
- 数据源读取
- 条件分支
- 循环执行
- OCR 节点调用
- LLM 节点调用

## 当前验证方式

- 运行 `test-plans/basic/` 下的基础 plan
- 运行 `test-plans/data-driven/` 下的数据驱动 plan
- 检查各 plan 包自己的 `output/` 日志结果

## AI 节点补充说明

- `ocr_image` 已经有本地可运行示例，可以直接验证 OCR sidecar 链路。
- `ocr_image` 目前建议至少验证两种输入：本地文件、页面元素截图。
- 当前这两种 OCR 输入链路都已完成示例验证。
- `llm_chat` 已补齐调用接口和示例，但实际是否稳定通过，还取决于模型网关和当前模型响应时间。
- `llm_extract_json` 依赖模型节点稳定性，建议在 OCR 与本地网关都稳定后再做回归验证。
