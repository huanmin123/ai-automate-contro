# 安装指南

```powershell
python -m pip install -e .
winget install --id BurntSushi.ripgrep.MSVC -e
rg --version
python -m playwright install chromium
```

建议使用 PowerShell 7 作为默认 shell。

AI 终端的渐进式文本搜索依赖 `ripgrep` 的 `rg` 命令，项目不使用 Windows 内置搜索做兜底。如果 `rg --version` 失败，先安装 `ripgrep`，或让用户确认后由助手执行上面的全局安装命令。安装后如果当前终端还找不到 `rg`，重新打开 PowerShell 7 再验证。
