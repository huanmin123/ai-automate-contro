# 桌面依赖安装说明

本文记录桌面控制线需要的软件、Python 包、系统组件、安装来源、安装路径、能力矩阵字段和验证命令。这里是开发/运行环境说明，不是 plan 教程；AI 写 plan 时仍读取 `handbook/`。

## 依赖分层

| 层级 | 依赖 | 当前用途 | 能力矩阵字段 |
| --- | --- | --- | --- |
| 必需 | Python 3.11+，推荐 3.13 | 项目运行、CLI、自检 | - |
| 必需 | PowerShell 7 | Windows 默认 shell、临时 WinForms 回归、UIAutomation 辅助脚本 | - |
| 必需 | 项目基础依赖 | plan 执行器、AI 终端、浏览器线和通用能力 | - |
| 桌面输入 | `pyautogui` | 键盘、鼠标、点击、拖拽、滚动、快捷键 | `dependencies.pyautogui` |
| 桌面剪贴板 | `pyperclip` | `desktop_input type_text method=clipboard`、长文本输入、剪贴板恢复 | `dependencies.pyperclip` |
| 截图 | `Pillow` / `PIL.ImageGrab` | 全屏、区域、窗口、控件截图 | `dependencies.Pillow.ImageGrab` |
| 图像定位 | `opencv-python` | `desktop_vision type=locate_image` 模板匹配 | `dependencies.opencv-python` |
| OCR | Tesseract OCR 可执行文件 | `desktop_vision type=locate_text` | `dependencies.tesseract` |
| OCR 语言包 | `eng.traineddata` | 英文 OCR，`--require-ocr` 强制项 | `dependencies.tessdata.eng` |
| OCR 语言包 | `chi_sim.traineddata` | 简体中文 OCR，中文桌面文字定位 | `dependencies.tessdata.chi_sim` |
| Windows 控件语义 | Windows UIAutomation、`ctypes`、PowerShell .NET | 窗口枚举、控件树、表格、树、菜单、滚动容器 | `capabilities.semantic.*` |
| Windows 回归夹具 | .NET WinForms | `desktop-components` 临时窗口、文件对话框、复杂控件自检 | - |
| 浏览器线 | Playwright Chromium | 浏览器执行线；桌面线不依赖它，但完整项目需要 | - |
| 工具链 | ripgrep `rg` | AI 终端渐进式文本搜索 | - |

`pyproject.toml` 中的 `desktop` extra 还声明了 `mss` 和 Windows `pywinauto`。它们属于桌面增强依赖：当前能力矩阵不把它们作为必需项，`desktop-env` 会报告是否安装，但 `--require-input/vision/ocr/ocr-zh` 不把它们作为门禁；后续截图后端、UIA backend 增强或问题定位时可以继续使用。

`pytesseract` 不是当前运行时必需项。当前 OCR 路径直接调用系统 `tesseract` 命令并解析 TSV 输出。

## Windows 安装

基础环境：

```powershell
python -m pip install -e .
```

如果本机没有 `winget`，从 ripgrep 发布页或现有运维软件源安装 `rg`，安装后验证：

```powershell
rg --version
```

### Tesseract OCR

官方入口：

- Tesseract 安装说明：https://tesseract-ocr.github.io/tessdoc/Installation.html
- Windows installer 来源：UB Mannheim Tesseract installer：https://github.com/UB-Mannheim/tesseract/wiki
- 语言包来源：tesseract-ocr/tessdata_fast：https://github.com/tesseract-ocr/tessdata_fast

当前 Windows 开发机安装记录：

| 项 | 值 |
| --- | --- |
| 安装目录 | `E:\Tesseract` |
| 可执行文件 | `E:\Tesseract\tesseract.exe` |
| 已加入用户 PATH | `E:\Tesseract` |
| 版本 | `tesseract v5.4.0.20240606` |
| installer | `tesseract-ocr-w64-setup-5.4.0.20240606.exe` |
| installer URL | `https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-w64-setup-5.4.0.20240606.exe` |
| 已有语言包 | `eng`、`osd`、`chi_sim` |
| 中文语言包 URL | `https://github.com/tesseract-ocr/tessdata_fast/raw/main/chi_sim.traineddata` |

复现安装：

```powershell
$installerUrl = "https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-w64-setup-5.4.0.20240606.exe"
$installerPath = Join-Path $env:TEMP "tesseract-ocr-w64-setup-5.4.0.20240606.exe"
Invoke-WebRequest -Uri $installerUrl -OutFile $installerPath
Start-Process -FilePath $installerPath -ArgumentList @("/VERYSILENT", "/NORESTART", "/SUPPRESSMSGBOXES", "/DIR=E:\Tesseract") -Wait -WindowStyle Hidden
```

如果静默安装器没有按预期写入目录，直接运行安装器并在界面里选择 `E:\Tesseract`。安装后以 `Test-Path E:\Tesseract\tesseract.exe` 为准。

安装简体中文语言包：

```powershell
$tessdataDir = "E:\Tesseract\tessdata"
New-Item -ItemType Directory -Force -Path $tessdataDir | Out-Null
Invoke-WebRequest `
  -Uri "https://github.com/tesseract-ocr/tessdata_fast/raw/main/chi_sim.traineddata" `
  -OutFile (Join-Path $tessdataDir "chi_sim.traineddata")
```

加入当前用户 PATH：

```powershell
$tesseractDir = "E:\Tesseract"
$userPath = [Environment]::GetEnvironmentVariable("Path", "User")
$parts = @()
if ($userPath) { $parts = $userPath -split ";" | Where-Object { $_ -ne "" } }
if (-not ($parts | Where-Object { $_.TrimEnd("\") -ieq $tesseractDir.TrimEnd("\") })) {
  [Environment]::SetEnvironmentVariable("Path", (($parts + $tesseractDir) -join ";"), "User")
}
$env:Path = "$tesseractDir;$env:Path"
```

已打开的终端、IDE、Codex 桌面进程或后台服务可能不会自动读取新的用户 PATH。重启对应进程后再验证；不重启时可以在当前 PowerShell 临时执行 `$env:Path = "E:\Tesseract;$env:Path"`。

### Tesseract 路径配置兜底

如果当前进程没有继承新 PATH，或不希望依赖系统 PATH，可以在集合级 `<plan-root>/config.json` 或当前 plan 包 `config.json` 写入：

```json
{
  "desktop": {
    "ocr": {
      "tesseract_path": "E:/Tesseract/tesseract.exe",
      "tessdata_dir": "E:/Tesseract/tessdata",
      "default_language": "eng"
    }
  }
}
```

配置优先级：plan 包 `config.json` 覆盖集合级 `<plan-root>/config.json`。未配置时再尝试 `AI_AUTOMATE_TESSERACT_PATH`、`AI_AUTOMATE_TESSDATA_DIR`、当前进程 PATH、Windows 用户/系统 PATH 注册表和常见安装目录。

当前仓库自检默认读取 `test-plans/config.json`；公开示例 plan 默认读取 `plans/config.json`；真实业务 plan 可在对应 plan 包里单独覆盖。

## macOS 安装

macOS 真机后续验收时按以下原则安装，不在 Windows 上模拟。

Python 依赖：

```bash
PIP_REQUIRE_VIRTUALENV=false python3 -m pip install --user --break-system-packages -e ".[desktop]"
python3 -m playwright install chromium
brew install ripgrep
```

Tesseract：

```bash
brew install tesseract tesseract-lang
tesseract --version
tesseract --list-langs | grep -E "eng|chi_sim"
```

如果不安装完整 `tesseract-lang`，也可以把所需 `.traineddata` 放到 Homebrew Tesseract 的 `tessdata` 目录；目录位置以 `tesseract --list-langs` 输出为准。

macOS 权限：

- Accessibility：窗口控制、控件读取、键鼠输入通常需要。
- Screen Recording：截图、屏幕 OCR、视觉定位通常需要。
- Automation：AppleScript/System Events 控制 App 时可能需要。

首次触发授权时，让用户在系统弹窗或系统设置里确认。授权对象通常是启动 Python/终端/打包 App 的那个进程；换启动主体后可能需要重新授权。

macOS 验证：

```bash
python3 main.py tool call inspect_desktop --args-json '{"platform_name":"auto","request_permissions":false,"include_windows":true,"include_elements":false,"max_windows":5}' --compact
python3 cplan.py self-check desktop-env
python3 cplan.py self-check desktop-components
python3 cplan.py self-check desktop-components --require-ocr
python3 cplan.py self-check desktop-components --require-ocr-zh
```

需要到 Mac 真机继续补齐 Retina/DPI、多显示器、AX 表格/树/菜单和系统文件对话框回归。

## 常见问题

### `tesseract` 找不到

检查 PATH：

```powershell
Get-Command tesseract -ErrorAction SilentlyContinue
[Environment]::GetEnvironmentVariable("Path", "User")
```

临时修复当前终端：

```powershell
$env:Path = "E:\Tesseract;$env:Path"
```

### `tessdata.eng=false`

说明 Tesseract 可执行文件可用，但英文语言包不可用或 `tessdata` 路径不对。先运行：

```powershell
tesseract --list-langs
```

如果没有 `eng`，重新安装 Windows installer，或把 `eng.traineddata` 放到 `E:\Tesseract\tessdata`。

### `tessdata.chi_sim=false`

说明中文包未安装。下载 `chi_sim.traineddata` 后放到 `E:\Tesseract\tessdata`，再重开终端或重新运行探测命令。

### `opencv-python=false`

安装桌面 extra：

```powershell
python -m pip install -e ".[desktop]"
python -c "import cv2; print(cv2.__version__)"
```

### `pyautogui=false` 或鼠标键盘不可用

安装桌面 extra 后重新验证。远程桌面、锁屏、权限受限、高权限窗口和安全桌面仍可能阻止真实输入。

### OCR 识别质量不好

优先缩小 `source_target`、`region` 或提高截图清晰度。英文用 `language="eng"`，中文混合用 `language="chi_sim+eng"`。如果命中不稳定，提高 `min_confidence` 并保留 `desktop_capture` 或 `desktop_vision` 的标注产物。
