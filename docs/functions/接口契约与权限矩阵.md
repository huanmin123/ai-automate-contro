# 接口契约与权限矩阵

## 计划文件契约

- `plan` 文件必须包含 `steps`
- `variables` 可选

## 配置文件契约

- 全局 plan 配置固定为 `plans/config.json`
- 局部 plan 配置固定为当前 plan 包根目录的 `config.json`
- 两者都必须是 JSON 对象
- 局部配置覆盖全局配置
- `config.variables` 必须是 JSON 对象

## 动作契约

- 每个动作必须有 `action`
- 与页面交互相关的动作通常需要 `browser`
- 与元素交互相关的动作通常需要 `selector`

## 当前边界

- 当前没有做权限系统
- 当前默认本地全权限执行
- 当前依赖本机 Playwright 浏览器环境
