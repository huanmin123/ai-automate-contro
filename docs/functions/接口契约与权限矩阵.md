# 接口契约与权限矩阵

## 计划文件契约

- `plan` 文件必须包含 `steps`
- `variables` 可选

## 配置文件契约

- 集合级 plan 配置固定为当前运行根 `plan.config.plan_roots` 下的 `<plan-root>/config.json`
- 局部 plan 配置固定为当前 plan 包根目录的 `config.json`
- 两者都必须是 JSON 对象
- 局部配置覆盖全局配置
- `config.json` 只保存配置，不保存 plan 变量；plan 变量写在 `plan.json.variables`

## 动作契约

- 每个动作必须有 `action`
- 与页面交互相关的动作通常需要 `browser`
- 与元素交互相关的动作通常需要 `selector`

## 当前边界

- 当前没有做权限系统
- 当前默认本地全权限执行
- 当前依赖本机 Playwright 浏览器环境
