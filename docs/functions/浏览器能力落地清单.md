# 浏览器能力落地清单

这个文档记录项目级浏览器能力现状与后续 backlog，不属于组件教学手册。

## 已优先落地

- 浏览器上下文配置：多浏览器类型、viewport、locale、timezone、user_agent、headers、proxy、权限、移动端基础参数、设备预设、HAR 录制、视频录制。
- 定位能力：普通 selector、iframe selector/name/URL/index 定位、role/text/label/placeholder/alt/title/test_id 语义定位。
- 元素能力：双击、右键、拖拽、强制点击、位置点击、触控 tap、触控 swipe、修饰键、文件选择器。
- 等待能力：load state、element state、JS 条件等待。
- 网络能力：request/response 捕获增强、响应体读取、route mock、abort、continue、headers、HAR 回放、WebSocket mock。
- 脚本与状态：evaluate、add_init_script、cookies、localStorage、sessionStorage。
- 观测能力：trace start/stop、coverage start/stop、console、pageerror、requestfailed、WebSocket 起始事件与帧采集、SSE 事件采集、WebRTC 事件采集、Service Worker 事件采集。
- 数据与断言：URL、title、frame 列表、bounding box、CSS、ARIA snapshot 提取；attribute、CSS、checked、enabled、visible、hidden、title 断言。

## 后续再做

- WebRTC 完整媒体轨道录制、Service Worker cache storage 内容导出等专项深挖能力。

## 推荐优先级

1. WebRTC 完整媒体轨道录制。
2. Service Worker cache storage 内容导出。
