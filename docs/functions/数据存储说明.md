# 数据存储说明

项目当前的数据主要分成四类：

## 1. 输入数据

- `examples/scenarios/basic/`
- `examples/scenarios/data-driven/`
- `examples/suites/smoke/`
- `examples/suites/regression/`
- `examples/data/`

## 2. 运行态数据

- 内存中的 `variables`
- 浏览器会话状态

## 3. 输出数据

- `output/plan-runs/`
- `output/suite-runs/`
- `result.json`
- `suite-summary.json`
- `suite-summary.md`
- `failure-screenshots/`

## 4. 文档数据

- `handbook/`
- `docs/`

## 外部数据源格式

当前支持：

- `json`
- `csv`
- `txt`
