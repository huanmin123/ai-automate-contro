# 数据存储说明

项目当前的数据主要分成四类：

## 1. 输入数据

- `test-plans/basic/`
- `test-plans/data-driven/`
- 每个 plan 包自己的 `resources/`

## 2. 运行态数据

- 内存中的 `variables`
- 浏览器会话状态

## 3. 输出数据

- 每个 plan 包自己的 `output/`
- `result.json`
- `failure-screenshots/`
- 截图、录屏、下载、HTML、JSON、CSV、TXT、storage state 和 OCR 临时截图
- 输出动作按组件分区写入，例如 `screenshots/`、`downloads/`、`html/`、`json/`、`csv/`、`text/`、`storage-states/`、`ocr-captures/`

## 4. 文档数据

- `handbook/`
- `docs/`

## 外部数据源格式

当前支持：

- `json`
- `csv`
- `txt`
