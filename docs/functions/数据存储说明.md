# 数据存储说明

项目当前的数据主要分成四类：

## 1. 输入数据

- 当前运行根 `plan.config.plan_roots` 中的 plan 包
- 每个 plan 包自己的 `resources/`

## 2. 运行态数据

- 内存中的 `variables`
- 浏览器会话状态

## 3. 输出数据

- 每个 plan 包自己的 `output/`
- `state.json`
- `result.json`
- `report.md`
- `run.log`
- `events.jsonl`
- 交互式运行的 `commands.jsonl`
- `failure-screenshots/`
- `failure-html/`
- `failure-page-state/`
- 截图、录屏、下载、HTML、JSON、CSV、TXT 和 storage state
- 输出动作按组件分区写入，例如 `screenshots/`、`downloads/`、`html/`、`json/`、`csv/`、`text/`、`storage-states/`

## 4. 文档数据

- `handbook/`
- `docs/`

## 外部数据源格式

当前支持：

- `json`
- `csv`
- `txt`
